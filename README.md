# Grace Engineering - Final Digital Experience 🚀

Welcome to the official digital storefront for **Grace Engineering**.

**🌐 Live Site:** [https://graceengineering.in](https://graceengineering.in)

This repository contains the final source code for the Grace Engineering website, engineered to showcase their premium manufacturing capabilities, advanced CNC/VMC machining, and ISO 9001:2015 certified infrastructure.

## 🌟 Key Features

- **Immersive 3D Experience**: Custom-built interactive 3D industrial gear on the Hero section powered by `three.js` and `@react-three/fiber`.
- **Premium UI & Animations**: Buttery smooth entrance animations, parallax tilt cards, and a vertical image accordion built with `framer-motion`.
- **Mobile-First Responsive Design**: Flawlessly optimized across all devices, ensuring the premium aesthetic scales perfectly to mobile screens.
- **Dynamic SEO Architecture**: Built on Next.js 14 (App Router) for blazing fast load times and optimal search engine visibility.
- **Serverless Form Integration**: Fully functional contact form linked directly to sales via FormSubmit API.

## 🛠 Tech Stack

- **Framework**: React 18 / Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **3D Rendering**: Three.js / React Three Fiber / React Three Drei
- **Icons**: Lucide React

## 🚀 Running Locally

To run the application on your own machine:

```bash
# 1. Install dependencies
npm install

# 2. Run the development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
